# 🗺️ Roadmap: Java Console Calculator

Данный документ описывает этапы развития консольного калькулятора на Java. Проект создается для освоения базового синтаксиса, работы с условными операторами, циклами и обработки пользовательских исключений.

## 🎯 Цель проекта
Создание отказоустойчивой программы-калькулятора, работающей в консоли, с поддержкой базовых математических операций, непрерывного цикла выполнения и защитой от некорректного пользовательского ввода.
